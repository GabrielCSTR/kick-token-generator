Gerador de Tokens para a API da Kick

Esta ferramenta permite gerar tokens de acesso OAuth para a API da Kick.tv de forma segura. Insira suas credenciais de aplicativo e selecione os escopos necessários para sua aplicação.

⚠️
Atenção à segurança
Nunca compartilhe seus tokens de acesso ou suas credenciais de aplicativo. Eles fornecem acesso à sua conta Kick e podem ser usados para realizar ações em seu nome.
Suas Credenciais de Aplicativo

Para obter suas credenciais, você precisa registrar um aplicativo no Painel de Desenvolvedores da Kick.

Selecione os Escopos Necessários
Visualize informações do usuário no Kick, incluindo nome de usuário, ID do streamer, etc.
Veja informações do canal no Kick, incluindo descrição do canal, categoria, etc.
Atualizar metadados de transmissão ao vivo para um canal com base no ID do canal
Envie mensagens de bate-papo e permita que bots de bate-papo postem em seu bate-papo
Leia o URL de fluxo e a chave de fluxo de um usuário
Assine todos os eventos do canal no Kick, por exemplo, mensagens de bate-papo, seguidores, assinaturas
Executar ações de moderação para moderadores
