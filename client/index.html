Token Generator for the Kick API

This tool allows you to generate OAuth access tokens for the Kick.tv API securely. Enter your application credentials and select the necessary scopes for your application.

⚠️
Security Notice
Never share your access tokens or application credentials. They provide access to your Kick account and can be used to perform actions on your behalf.
Your Application Credentials

To get your credentials, you need to register an application in the Kick Developer Dashboard.

Select Required Scopes
View user information on Kick, including username, streamer ID, etc.
View channel information on Kick, including channel description, category, etc.
Update live stream metadata for a channel based on channel ID
Send chat messages and allow chatbots to post in your chat
Read a user's stream URL and stream key
Subscribe to all channel events on Kick, e.g., chat messages, followers, subscriptions
Execute moderation actions for moderators
